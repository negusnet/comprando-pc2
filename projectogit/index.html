Palabras Definicion
Serie X

El procesador de la serie X tiene hasta 18 núcleos. Está pensado para necesidades de software extremadamente exigentes, como los juegos de gama alta o la edición de vídeo.
i9

El i9 ofrece hasta ocho núcleos. Este procesador es útil para las necesidades de juego y transmisión de vídeo por encima de la media.
i7

El Core i7 ofrece un rendimiento superior a la media para la informática general. Es bueno para la multitarea en ordenadores portátiles o de sobremesa que necesitan altas velocidades de reloj.
i5

Para aquellos con necesidades informáticas medias, el i5 les servirá bien. Puede manejar todas las tareas informáticas del día a día de manera eficiente. Incluso funciona bien para las necesidades medias de juego.
i3

El Core i3 es el miembro más económico de la familia Core. Ofrece un rendimiento decente, pero no está hecho para los juegos de gama alta o para aquellos que necesitan una capacidad multitarea significativa.
